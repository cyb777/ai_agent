# ai_agent 项目文件清单

## 本人独立完成

| 文件 | 说明 |
|------|------|
| `.env`（本地未上传）/ `.env.example` | 数据库、邮件、模型连接配置与示例模板 |
| `app/main.py` | FastAPI 入口，生命周期，CORS，路由注册 |
| `app/ai/model/model.py` | MyModel 单例模式，统一管理大模型实例 |
| `app/ai/tool/mysql_tool.py` | MySQL 查询工具，完善 6 张表 docstring |
| `app/ai/tool/send_email_tool.py` | QQ 邮箱 SMTP 工具，Logger 集成，环境变量配置化 |
| `app/ai/agent/system_agent.py` | 登录验证码智能体，mysql_tool + send_email 双工具协作 |
| `app/ai/agent/sql_question_agent.py` | SQL 问答智能体，异步流式输出，InMemorySaver 对话记忆 |
| `app/ai/agent/echarts_agent.py` | ECharts 图表智能体，查询数据库自动生成图表 JSON |
| `app/ai/agent/anlyze_agent.py` | 数据分析智能体，三步流程（查询→表格+分析→图表） |

## 来自开源模板

| 文件 | 说明 |
|------|------|
| `app/utils/logger.py` | 日志工具类 |
| `app/api/chat/chat_router.py` | 聊天路由（SSE 流式响应 + 智能体路由） |
| `app/api/system/system_router.py` | 系统路由（登录注册接口 + Redis 验证码） |
| `app/api/schema/login_schema.py` | 登录注册参数模型 |
| `app/ai/schema/mysqlSchema.py` | SQL 参数模式 |
| `app/ai/schema/emailSchema.py` | 邮件参数模式 |
| `app/ai/schema/emailResponse.py` | 邮件响应格式 |
| `app/ai/schema/ectarts_response.py` | 图表响应格式 |
| `app/ai/schema/anlyze_response.py` | 数据分析响应格式 |
| `app/` 下所有 `__init__.py` | 包标识文件 |
